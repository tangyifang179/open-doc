# 表单嵌入

你可以将表单嵌入到自己的网页博客或微信公众号中。相关设置和参数在表单 「发布」 —— 「嵌入表单」 中获取。

![](https://help-assets.jinshuju.net/assets/file/8983/1%E5%8F%91%E5%B8%83.png)

## 嵌入微信公众号

![](https://help-assets.jinshuju.net/assets/file/5173/5.png)

> 注意：需要认证过的服务号才支持子菜单配置。

可<a href="https://help.jinshuju.net/articles/wechat" target="_blank">点击此处</a>查看更详细的图文指导。

## 在网页中嵌入悬浮按钮

通过将代码粘贴至网页 HTML 期望的位置。你可以在网页中嵌入悬浮式按钮，用户点击后可在新页面打开表单。

![](https://help-assets.jinshuju.net/assets/file/5177/6.png)

## 在网页中嵌入填写页面

支持 script 和 iframe 两种嵌入方式。

![](https://help-assets.jinshuju.net/assets/file/5178/7.png)

在网页中嵌入表单时，系统默认表单 background 为 white ，若需要显示网页颜色，需在嵌入的 script 中将 background 改成 transparent 。

![](https://help-assets.jinshuju.net/assets/file/3107/__.png)

![](https://help-assets.jinshuju.net/assets/file/3108/________.png)

> 注意：因微信官方限制，开启<a href="https://help.jinshuju.net/articles/wechat-personal-info" target="_blank">微信收集信息</a>或<a href="https://help.jinshuju.net/articles/wxpay" target="_blank">微信支付</a>后，会导致表单嵌入失效，请关闭后再使用表单嵌入功能。

## 定制参数

* 链接中的 banner 可以设置为 show 或 hide ，对应显示页眉或隐藏页眉。
* 链接中的 background 可以设置为 white 或 transparent ，对应为白色背景或透明背景。
