# 概览

金数据 API v1 是一套基于 HTTP RESTful 风格的接口能力，使用 API Key / Secret 进行安全认证。你可以通过 API 读取表单结构、查询表单数据、创建和更新数据，并将金数据接入业务系统、自动化流程或内部工具。

## 适合什么场景

API v1 适合需要通过程序读取、创建、修改或同步金数据表单与数据的系统集成场景。

你可以用 API v1：

- 将表单数据同步到 CRM、工单系统、运营后台或数据仓库
- 在业务系统中读取表单结构和提交数据
- 通过程序创建、更新或删除表单数据
- 查询当前账号、企业成员和套餐信息

## 快速开始

如果你是第一次接入 API v1，建议按下面的顺序开始：

1. 查看 [认证方式](/api_v1/authentication/)，获取 API Key 和 API Secret
2. 查看 [接口列表](/api_v1/all/)，找到你需要调用的接口
3. 查看 Schema，了解表单结构和数据结构
4. 查看 [状态码](/api_v1/status_code/)，了解常见错误和处理方式
5. 查看 [请求速率](/api_v1/request_rate/)，确认调用限制

> 金数据 API v1 的可用能力会根据账号套餐和权限有所不同。接入前，请先确认当前账号是否已开启对应 API 权限。
