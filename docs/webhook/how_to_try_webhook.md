# 如何测试 Webhook

在你正式使用金数据表单 Webhook 功能前，你可能想要测试一下表单 webhook，可以使用以下方式来完成。

## 使用 jsj-api 工具

[https://jsj-api.jinshujuapp.com/webhook_endpoints/new](https://jsj-api.jinshujuapp.com/webhook_endpoints/new)

1. 打开 jsj-api webhook 测试工具
2. 生成一个新的 Webhook 测试地址
3. 复制生成的 Webhook 测试地址，在你需要测试的表单上配置地址
4. 提交一条新的数据
5. 在 jsj-api webhook 测试工具，刷新页面，查看推送结果

> 注意：jsj-api 工具并不是金数据官方维护的产品，请勿适用于真实应用场景
