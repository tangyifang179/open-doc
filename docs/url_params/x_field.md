# 金数据 URL 传参 - 扩展属性

## 概述

在表单发布出去的 URL 地址上，加入 `x_field_1` 参数以及你期望的值，就可以使用 扩展属性 URL 传参。

格式： `https://jinshuju.net/f/TOKEN?x_field_1=$VALUE$`

* 不需要在金数据中提前配置
* 传入单个值，以原始字符串的格式保存
* 无法校验是否被篡改
* 不同的表单可以传入同样的业务
* 金数据系统中按照文本字段使用（比如数据展示、Webhook）

> 如果 VALUE 中含有特殊字符，则需要进行 URL Encode 转义

## 如何配置

### 在系统中配置

进入表单的 「发布」 - 「跟踪数据来源」 - 「设置扩展属性」

可以在「扩展值」中，输入期望的值，点击「生成地址」既可自动生成对应的表单 URL


### 手工配置或者编程配置

可以直接在表单的 URL `https://jinshuju.net/f/TOKEN` 后面加入 `?x_field_1=VALUE` 即为最终发布的 URL
