# 金数据开放平台

金数据开放平台，提供了强大灵活的系统集成能力。你可以将你的系统或者第三方系统，与金数据链接起来，实现更为强大的系统综合应用。

目前金数据开放平台提供的能力包括：

1. MCP Server

金数据 MCP（Model Context Protocol）Server，允许 AI 助手（如 Claude、ChatGPT 等）通过标准化的 MCP 协议，直接与金数据平台进行交互。AI 助手可以帮助用户管理表单、查询和操作数据，极大地提升工作效率。

2. API (v1)

金数据 API 提供了多个接口，使用者可以通过 API 获取表单信息、数据信息，也可以通过 API 来创建或者修改数据。

2. Webhook

金数据 Webhook 提供了数据推送能力。当表单收集到新的数据，或者数据发生变更时，金数据可以主动推送数据到你的服务器。

3. URL 传参

在你向填表人推送表单链接时，可以在 URL 上传入数据，这些数据会被自动获取到最终金数据的表单数据上。

4. 表单嵌入

可以在你的公众号、网页、App 中，嵌入金数据表单，收集数据。
